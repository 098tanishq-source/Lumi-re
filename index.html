Project Overview

Behind the Design

A breakdown of the design thinking, strategy, and technical execution behind Lumière's digital presence.

01
Goal
Build a premium restaurant website that converts visitors into reservations. The primary goal was to create an emotional connection before the customer even visits — communicating elegance, quality, and trust through design alone.
ConversionTrustBrand
02
Target Audience
Urban professionals, couples, and corporate clients aged 28–55. People who value experiences over price and make decisions based on visual credibility and social proof. Mobile-first — most traffic comes from Instagram discovery.
UrbanMobile-firstPremium
03
Design Direction
Luxury-minimal. Cormorant serif for editorial elegance paired with Manrope for clean readability. Alternating dark/cream sections create visual rhythm. Gold accent — restrained, never overused.
Luxury-MinimalSerif + Sans
04
Responsive Strategy
Three breakpoints: 1024px (tablet), 768px (mobile), 480px (small mobile). Container-based layout prevents horizontal overflow. No fixed widths — everything fluid with max-width: 1200px containers.
Mobile-first3 Breakpoints
05
Tools Used
Pure HTML5 + CSS3 + Vanilla JS. No frameworks, no dependencies. Google Fonts for typography. Unsplash for photography. WhatsApp Business API for reservations. Single-file deployment for GitHub Pages.
HTML5CSS3Vanilla JSWhatsApp API
06
Key Interactions
Smooth scroll navigation. Scroll-triggered reveal animations (subtle, single-direction). WhatsApp-powered reservation form with live validation. Functional drink order buttons. Animated counters. Random page entry animation per reload.
Scroll RevealWhatsAppValidation
